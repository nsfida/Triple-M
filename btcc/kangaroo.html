How it Works

  1. What G means: G is the known Generator point of the secp256k1 elliptic curve. It is the universal starting point for Bitcoin keys.
  2. What public key means: The public key P is mathematically derived by multiplying the generator by the private key: P = x * G. Finding x from P is the Discrete Logarithm Problem.
  3. What tame kangaroo does: Each worker starts an independent tame walk from b + offset and executes a deterministic, pseudo-random walk based on point coordinates. Periodically, it drops "Distinguished Points" (markers) into the shared coordinator map.
  4. What wild kangaroo does: Each worker starts an independent wild walk from P + offset*G. It uses the exact same deterministic walk logic. If its path intersects any tame trail, the main thread sees the shared marker collision.
  5. What collision means: A collision occurs when Wild hits a Tame marker (or vice versa). Because their absolute curve positions are perfectly equal at that moment, the coordinator solves x = tame_start + d_T - d_W.
